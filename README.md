# CaptureStream
ストリーミングで公開されているNHKラジオ語学講座のダウンロードを自動化するためのツールです。  開発が中断している https://osdn.net/projects/capturestream/  のForkです。 正式に開発を引き継いでいない非公式版です。ファイル自体のダウンロードにはffmpegを利用します。
